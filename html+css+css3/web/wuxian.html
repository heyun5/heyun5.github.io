无限锤被评为广州市年度网络科技放心消费创建先进单位
2015-10-5
中科龙源网络科技有限公司位于江苏无锡新区国家传感网示范区国际创新园，是一家由中科院物联网研究发展中心参股，以物联网云电商平台服务为主营业务的高科技企业。如今，中科龙源网络科技有限公司再次布局华南，进驻光电深圳。7月13日，中科龙源网络科技有限公司和中科新星科技有限公司达成成战略伙伴关系，这标志着，中科龙源在华南地区的业务布局已基本完成。 [更多]
